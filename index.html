Thanks for using SparQLed.

  • Live Demo on the Health Care Life Science datasets
  • Instructions for installing SparQLed are available in the Downloads section

Distribution Content

This distribution includes 6 modules:

  • sparql-editor-client: the editor User Interface;
  • recommendation-servlet: the servlet for providing query element recommendations;
  • sesame-sparql-queryparser: Sesame SPARQL query parser extended for the SparQLed use case;
  • sesame-backend: Utility classes for operating on SPARQL endpoint through Sesame;
  • analytics-commons: Utility classes used in the Data Graph Summary computation;
  • sparql-summary: an SPARQL-based Data Graph Summary Computation.
